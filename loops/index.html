For Loop

  • Mainly used for iterating between arrays and numerical loops

  • Doesnot return any value but can use async, break and continue.

                          //process or instructions to what to do
                      
                          console.log("i looped ",loop," time");
          }

While Loop

  • Usually used when user or program doesnot know how many times to loop but know the expected ressult
  • Does not return any data but we ca use break/continue and is async friendly.
  • While loops checks condition first and then run the loop , if condition is false it doesnot run the loop.
    •                 while   (condition)  {


                                // processing or instructions

                                //update condition

                                count =0;
                                
                                count++;

                        console.log("while repeated here" )
                    }

for ..in Loop

  • just for iterating keys/properties[like in dictionaries] of an object
  • not recomended for arrays (can include inherited keys)
  • Here is a syntax example of how we can use loops.
    •                 const person = { name: "Alice", age: 25 };
                for (const key in person) {
                        console.log(key + ": " + person[key]);
                    }

forEach() Loop

  • Iterating arrays with access to value, index, and original array.
  • Cannot use break or continue.
  • Good for clean inline loops.
    •                 const nums = [1, 2, 3];
                nums.forEach(function (num, index) {
                console.log(index, num);
                });
